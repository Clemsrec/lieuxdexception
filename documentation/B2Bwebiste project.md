Prompt pour Agent IA - Site Catalogue B2B Groupe Riou
Contexte du Projet
Créer un site catalogue B2B professionnel pour le Groupe Riou présentant une collection de 5 lieux événementiels en France, optimisé pour deux cibles principales : événements professionnels et mariages.
Stack Technique

Framework : Next.js 15+ (React)
Hébergement : Google Firebase
Base de données : Firestore
Langages : TypeScript recommandé
Styling : Tailwind CSS

Fonctionnalités Clés
Pages Principales

Accueil : Présentation du Groupe + mise en avant collection
Catalogue interactif : 5 lieux avec système de filtres (capacité, localisation, type, budget)
Événements B2B : Page dédiée séminaires/conférences
Mariages : Page dédiée réceptions privées
Contact : 3 types de formulaires qualifiés
Mentions légales + CGV
Politique de confidentialité RGPD

Système de Filtrage

Capacité d'accueil
Localisation géographique
Type d'événement (B2B/mariage)
Fourchette budgétaire
Tableau comparatif des 5 lieux

Formulaires de Contact (3 types)

Formulaire B2B détaillé : entreprise, type événement, nb participants, dates, budget
Formulaire mariages : couple, date souhaitée, nb invités, prestations
Formulaire contact rapide : nom, email, téléphone, message

Intégration CRM Odoo

Envoi automatique des leads vers Odoo via webhooks
Mapping des champs personnalisés
Notifications email automatiques
Stockage backup dans Firestore

Multilingue (6 langues)

Français (défaut)
Anglais, Espagnol, Allemand, Italien, Portugais
Détection automatique langue navigateur
URLs SEO-friendly par langue (/en/, /es/, etc.)

Back-Office Admin

Dashboard avec statistiques temps réel
Google Analytics 4 intégré
Google Search Console intégré
Gestion contenu catalogue (textes/photos)
Consultation leads avec export CSV
Gestion multi-utilisateurs
Métriques : trafic, conversions, positions Google

Politique de confidentialité auto-générée
Droits utilisateurs (accès, rectification, oubli, portabilité)
